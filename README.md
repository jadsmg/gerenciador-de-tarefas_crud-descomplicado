# API de Gerenciamento de Tarefas

API simples de gerenciamento de tarefas desenvolvida com Python e Flask.
A aplicação permite criar, listar, atualizar e remover tarefas por meio de endpoints REST.

### Tecnologias
- Python
- Flask
- Pytest
### Funcionalidades
- Criar tarefas
- Listar tarefas
- Atualizar tarefas
- Excluir tarefas
- Testes automatizados com Pytest

### Instalação
```bash
git clone https://github.com/xandsmg/gerenciador-de-tarefas_crud-descomplicado
cd gerenciador-de-tarefas_crud-descomplicado
pip install -r requirements.txt
```
### Executando o projeto
```bash
python3 app.py
```
### Executando os testes
```bash
pytest tests.py -v
```
